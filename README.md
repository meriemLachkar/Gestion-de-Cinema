# Gestion de Cinema

Projet de gestion d'un cinema permettant l'administration des films,
des salles et des séances de projection à l'aide d'une base de données MySQL.

## Fonctionnalités
- Gestion des films
- Gestion des salles
- Gestion des séance

## Technologies
- MySQL
- SQL
- UML
- Git
